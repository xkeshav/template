# Template Repository

This is [readme file][readme] of template repository; based on typescript and node.

## Pre-requisite

- Node 18+
- Typescript v 5.0+
- A Browser
- an IDE ( VS Code Preferred)

## How to start

- Navigate to the repo from terminal
- Install dependencies using below command

```bash
> npm install
```

then

```sh
> npm run start
```
